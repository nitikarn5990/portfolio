Valve ARG test subject thank you whiteboard

2011

http://valvearg.com/thankyou/whiteboard/

Marketing the launch of a highly anticipated game Portal 2, Valve cooperated with indie game developers created an ARG. (Alternate Reality Game) The ARG starts on 1 April 2011. On that day, various indie games have been updated. Players found that new, out of place stuff have been added to the games that could form cryptic clues that ultimately leads to GLaDOS@Home.

During the ARG, events were recorded in the ValveARG wiki. After the ARG have ended, one of the key player decided to ask the community to create a thank you page. Few days have passed with no news from the team. On April 22nd 2011, a countdown appeared on ValveARG.com. Presses guessed it was a countdown to a DLC. After the countdown ended, people start flooding the message board with thank you messages to Valve. Over 1,722 people have shown their gratitude on the message board. The thank you messages were read by the indie developers and Valve.

  • Technology: Steam API, OpenID, HTML5 Audio, CSS3
  • Language: PHP

Noke

2011

https://github.com/whs/Noke

Karaoke is a popular Thais pastime. So there're softwares for playing MIDI karaoke on Windows. However, the 3 file MIDI format originated from Nick Karaoke is not supported on Linux karaoke players so I made this web-based application to provide this feature.

Noke stands for Node+Karaoke. It uses midicore library to play MIDI on the host computer, and a Nodejs server polls midicore then broadcast it to WebSocket clients. The client then retrieve parsed lyric and cursor file and display the correct line and position according to the data received.

  • Technology: Node.js, WebSocket, UDP, CSS3
  • Language: JavaScript

OvzCP

2010

Second prize winner, National Software Competition 2011

http://whs.whs.in.th/ovzcp/

OvzCP is a web-based control panel for OpenVZ, a virtualization system for Linux. OvzCP is aimed to be user friendly by supporting internationalization and visualizations. OvzCP does not reinvent the wheels, but integrate many existing services such as Munin and Varnish.

One of the most awesome feature of OvzCP is its mobile site. In the mobile site users can change their Varnish configuration and restart their container. Users also can access to their data using OvzCP API.

OvzCP won the second prize from 13th National Software Competition and currently participating in Thailand ICT Award 2011

The server you're accessing right now is controlled by OvzCP and its Varnish integration.

  • Technology: OpenVZ, Varnish, jQuery Mobile, OpenID
  • Language: Python, Shell Script

DEngine

2009

http://whs.whs.in.th/dengine/

A Drupal RPG game engine. DEngine is the engine that powers DarkWorld, one of my scrapped project. DEngine is no longer maintained and now becomes open source. Consists of mostly JavaScript code and few PHP files to provide data persistent and anti-cheat.

DEngine, in fact is not a Drupal module, but it does use Drupal's user, hook, action, word filter and database system. DEngine requires a DEngine integration module to be installed in Drupal which provides database schema and anti-cheat mechanism.

  • Technology: CSS3, AJAX push, Drupal module
  • Language: PHP, JavaScript
